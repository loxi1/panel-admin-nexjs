# 🚀 Next.js Admin Dashboard Template

A modern and reusable admin dashboard template built with Next.js, designed for developers who need a clean and scalable starting point for web applications.

---

## 🌐 Live Demo

https://panel-admin-nexjs.vercel.app

---

## ✨ Features

* 📊 Dashboard UI ready to use
* ⚡ Fast and optimized with Next.js
* 🎨 Styled with Tailwind CSS
* 🔌 API-ready structure
* 📱 Fully responsive design
* 🧩 Modular and reusable components

---

## 🛠 Tech Stack

* Next.js
* React
* Tailwind CSS
* Node.js

---

## 📦 Installation

```bash
git clone https://github.com/loxi1/panel-admin-nexjs.git
cd panel-admin-nexjs
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```

---

## 🚀 Deployment

This project can be easily deployed using Vercel:

```bash
npm install -g vercel
vercel
```
